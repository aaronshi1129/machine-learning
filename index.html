Understanding AI Tasks

Classification

Predicting a categorical label or class for input data. It is task-driven supervised learning.

Example: Spam detection, image recognition

Regression

Predicting a continuous numerical value based on input features. It is task-driven supervised learning.

Example: Price prediction, temperature forecasting

Clustering

Grouping similar data points together without predefined labels. It is data-driven unsupervised learning.

Example: Customer segmentation, pattern discovery
